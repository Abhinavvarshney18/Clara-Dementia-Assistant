# Clara Dementia Assistant

## Description
This project is a backend + frontend system for assisting dementia patients.

## Features
- Authentication system
- Patient management
- Reminder system
- AI-based assistance

## Tech Stack
- Node.js
- Express.js
- JavaScript
- MongoDB / SQLite

## Setup
1. Clone repo
2. Run npm install
3. Add .env file
4. Run npm start
